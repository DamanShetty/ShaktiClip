ShaktiClip is a concept-level IoT safety system designed to provide instant SOS alerts and real-time location tracking, aimed primarily at enhancing women‚Äôs safety in remote and low-network areas. The system is built as a software-based prototype and simulation, demonstrating a complete end-to-end emergency response pipeline.

The project includes a simulated wearable device workflow, a backend alert-processing server, and a real-time guardian dashboard. When an SOS is triggered (manually or via simulated fall detection), the system captures location and sensor data, sends it to the backend, logs the alert, and displays it live on a monitoring dashboard with map integration.

This project was developed as part of a National-Level Hackathon (Innovex, NMAMIT) and was shortlisted among the Top 20 teams, based on concept strength, system architecture, and working simulation‚Äîwithout a physical hardware prototype.
